## 🧬 WHAT IS AGENTIC ALIEN?

> *"The first fully autonomous, self-learning DeFi empire that hunts alpha, promotes itself, and never sleeps."*

It combines a **self-adapting DeFi bot** (Ralph), **AI-powered token advertising** for three live pump.fun contracts, a **web-search-enhanced copywriting engine**, and a complete **gasless relayer** — all unified under one dashboard built for the next civilization of on-chain intelligence.

## 🚀 LOCAL SETUP

```bash
# Clone the repo
git clone https://github.com/imfromfuture3000-Android/agentic-ancient-alien.git
cd agentic-ancient-alien

# Install dependencies
npm install

# Set your environment variables (see below)
cp .env.example .env

# Start the app (frontend + backend on port 5000)
npm run dev
```

### Required Environment Variables

```bash
# Solana RPC
HELIUS_API_KEY=           # https://helius.dev
ALCHEMY_API_KEY=          # https://alchemy.com
MORALIS_API_KEY=          # https://moralis.io

# AI
OPEN_API_KEY=             # https://platform.openai.com

# Database
DATABASE_URL=             # PostgreSQL connection string

# Wallet
SOLANA_TREASURY=          # Your treasury wallet address
VITE_TREASURY_WALLET=     # Same, exposed to frontend
VITE_HELIUS_RPC_URL=      # Helius devnet URL for wallet adapter
VITE_REOWN_PROJECT_ID=    # WalletConnect project ID
```
