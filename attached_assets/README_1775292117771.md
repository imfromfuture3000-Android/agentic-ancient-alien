# SelfAdapting Agent - Solana Edition

This project implements a Node.js/TypeScript self-adapting LLM agent framework integrated with Solana RPC providers (QuickNode) and Helius.

**Important: never commit secrets to git.** Use `.env` or a secrets manager.

## Quickstart

1. Copy `.env.example` to `.env` and fill in your endpoints and API keys.
   ```
   cp .env.example .env
   # edit .env and add your keys
   ```

2. Install dependencies
   ```
   npm install
   ```

3. Run the agent in dev mode
   ```
   npm run dev
   ```

## What’s included
- `src/agent.ts` — main agent runner (self-edit collection + SFT stub)
- `src/solanaClient.ts` — QuickNode + Helius client wrappers (read-only)
- `src/mpcManager.ts` — MPC helper (local stub; integrate your MPC provider)
- `src/utils/logger.ts` — simple logger
- `.env.example` — placeholders for endpoints and API keys

## Security notes
- Keep `.env` local and out of version control.
- Use hardware wallets / HSM for signing; this project does not store private keys.
- Use Helius and QuickNode with rate-limits in mind.
